│   ├── 🏠 index.html        # Main website
│   ├── 📡 offline.html      # Offline experience page

## 🚀 Getting Started:

### 💻 Local Development:
```bash
# Clone the repository
git clone https://github.com/tusharbasak97/website.git

# Navigate to project directory
cd website

# Serve locally using Python
python3 -m http.server 8000

# Open in browser and enjoy full offline functionality
# Visit: http://localhost:8000
```

### 🧪 Testing Features:
```bash
# Test complete offline functionality
http://localhost:8000/test-complete-offline.html

# Test 404 error handling
http://localhost:8000/test-404.html

# Test offline experience
http://localhost:8000/test-offline.html
```
